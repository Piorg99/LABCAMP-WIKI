# Order Processing Workshop

This repository contains a simple Python application for processing customer orders.

## Project structure
- `models/`: domain entities
- `repositories/`: in-memory persistence
- `services/`: business logic
- `utils/`: validation and formatting helpers

## Main flow
1. Validate order
2. Calculate subtotal
3. Apply discount rules
4. Save processed order
5. Print summary
